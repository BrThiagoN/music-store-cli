# 🎸 Music Store CLI

Um projeto simples em Python de uma loja de instrumentos musicais rodando no terminal.
O sistema permite cadastrar instrumentos, listar produtos e navegar por um menu interativo.

Perfeito para treinar:

* Estruturas de dados
* Funções
* Organização de código
* Manipulação de listas
* Entrada de usuário
* Separação de arquivos Python
* Lógica de menus CLI

---

## 📁 Estrutura do Projeto

```bash
📦 music-store-cli
 ┣ 📜 Main.py
 ┣ 📜 data.py
 ┗ 📜 README.md
```

## ⚙️ Funcionalidades

✅ Cadastro de instrumentos
✅ Listagem de instrumentos
✅ Menu interativo no terminal
✅ Armazenamento temporário em memória

---

## 🧠 Tecnologias Utilizadas

* Python 3

## 🚀 Como Executar

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/music-store-cli.git
```

Entre na pasta:

```bash
cd music-store-cli
```

Execute o projeto:

```bash
python Main.py
```
